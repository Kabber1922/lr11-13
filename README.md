# Образовательная платформа - Сайт курсов

Десктопная версия сайта курсов, созданная на основе мобильных макетов из Figma. Сайт включает в себя систему авторизации, каталог курсов, детальную информацию о курсах и видео-плеер.

## Особенности

- ✅ **Flexbox и Grid Layout** - Использованы современные CSS технологии для адаптивной верстки
- ✅ **CSS Calc()** - Динамические вычисления размеров для адаптивности
- ✅ **Gulp сборка** - Автоматизация процессов разработки (опционально)
- ✅ **SCSS** - Препроцессор для удобной работы со стилями
- ✅ **Адаптивный дизайн** - Корректное отображение на всех устройствах
- ✅ **Интерактивность** - JavaScript функциональность для всех элементов

## Быстрый запуск (Демо версия)

Для быстрого просмотра сайта просто откройте файл `index.html` в браузере. Все страницы готовы к использованию без дополнительной настройки.

## Установка и запуск с Gulp

1. **Установка зависимостей:**
   ```bash
   npm install
   ```

2. **Запуск в режиме разработки:**
   ```bash
   npm run dev
   ```
   Откроется браузер с live-reload на `http://localhost:3000`

3. **Сборка проекта:**
   ```bash
   npm run build
   ```

4. **Отслеживание изменений:**
   ```bash
   npm run watch
   ```

## Страницы сайта

### 1. Главная страница (`index.html`)
- Загрузочный экран с анимированной лампочкой
- Приветственное окно с кнопками входа и регистрации
- Формы авторизации с валидацией
- Главное меню с каталогом курсов
- Поиск по курсам

### 2. Страница курсов (`courses.html`)
- Полный каталог всех доступных курсов
- Поиск и фильтрация
- Адаптивная сетка курсов

### 3. Детали курса (`course-details.html`)
- Информация о курсе с рейтингом и статистикой
- Превью видео
- Табы: Обзор, Информация, Сертификат
- Материалы для скачивания
- Информация о преподавателе

### 4. Видео курс (`course-video.html`)
- Видео плеер с контролами
- Транскрипт с временными метками
- Функции скачивания и поделиться

## Технологии

- **HTML5** - Семантическая разметка
- **SCSS** - Препроцессор CSS с переменными и миксинами
- **Flexbox** - Для гибкой компоновки элементов
- **CSS Grid** - Для создания сеток курсов
- **CSS Calc()** - Для динамических вычислений размеров
- **JavaScript ES6+** - Современный JavaScript
- **Gulp** - Автоматизация сборки

## CSS особенности

### Flexbox использование:
- Навигация и хедер
- Карточки курсов
- Формы авторизации
- Видео контролы

### Grid использование:
- Сетка курсов на главной странице
- Сетка курсов на странице каталога
- Адаптивные колонки

### Calc() использование:
- Адаптивные размеры контейнеров
- Динамические отступы
- Адаптивные размеры шрифтов
- Высота хедера

## JavaScript функциональность

- **Авторизация** - Вход и регистрация с валидацией
- **Поиск** - Фильтрация курсов в реальном времени
- **Табы** - Переключение между разделами курса
- **Видео плеер** - Воспроизведение и контролы
- **Уведомления** - Система уведомлений для пользователя
- **Адаптивность** - Обработка изменения размера окна

## Адаптивность

Сайт адаптирован для:
- **Десктоп** (1200px+)
- **Планшет** (768px - 1199px)
